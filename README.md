# 🌱 OptiCrop - Smart Agricultural Production Optimization Engine

OptiCrop is an AI-powered agricultural decision support system that helps farmers choose the most suitable crop based on soil nutrients and environmental conditions.

---

## Features

- 🌾 Crop Recommendation
- 🌱 Crop Suitability Analysis
- 📊 Prediction History
- 👨‍🌾 Farmer Dashboard
- 🔐 User Authentication
- 🤖 Machine Learning Prediction
- 📈 Agricultural Analytics

---

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- FastAPI
- SQLAlchemy
- JWT Authentication

### Database
- PostgreSQL

### Machine Learning
- Python
- Scikit-learn
- Pandas
- NumPy

---

## Input Parameters

- Nitrogen (N)
- Phosphorous (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall

---

## Output

- Recommended Crop
- Prediction Confidence
- Suitability Analysis
